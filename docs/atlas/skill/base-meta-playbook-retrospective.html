meta/playbook-retrospective

Use at the end of a coding session to capture playbook-worthy learnings before they fade. Analyzes the current Claude Code or Codex session log, extracts patterns (recurring corrections, new techniques, repeatable workflows), and drafts proposal markdown files in ~/.playbook-proposals/ that can later be promoted into the coding-agents-playbook repo. Manual-trigger only.
